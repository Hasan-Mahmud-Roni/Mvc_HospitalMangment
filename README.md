Project Name: Hospital Management System

Hospital Management System is a web application that allows patients to register, log in, create their personal details, and schedule appointments with one or more doctors.

 Technologies Used

- Backend: .NET Framework with ASP.NET MVC
- Frontend: HTML, CSS, JavaScript
- Database: MS SQL Server


 Prerequisites

- Visual Studio for backend development.
- MS SQL Server for storing patient and appointment data.

 Getting Started

Follow these steps to set up and run the project:

 Backend (.NET Framework / ASP.NET MVC)

1. Open the solution file Hospital_Management_Systems.sln in Visual Studio.
2. Build and run the project.

 Frontend (HTML/CSS/JavaScript)


 Features

- User Registration and Login: Patients can register and log in to the system.


- **Appointment Booking**: Patients can schedule appointments with one or more doctors.


 Usage

1. Register as a patient.
2. Log in to your account.
3. Create and manage your patient profile, including medical history.
4. Schedule appointments with doctors.
